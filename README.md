# The GOAT Playbook

The complete resource hub for G.O.A.T. Leads insurance agents. Scripts, templates, objection handling, and everything you need to work your leads and close deals.

## Features

- **11 Sales Scripts** - FEX, Mortgage Protection, IUL, Veterans, and Aged Lead openers
- **Knowledge Base** - Speed to Lead, voicemail strategies, text/email templates, objection handling, follow-up cadences, and more
- **Objection Handling** - Proven rebuttals including GOAT-specific objections ("I didn't fill out this form", "it said free")
- **Copy Buttons** - One-click copy for individual sections or full scripts
- **Search** - Find any content across all scripts and articles instantly
- **Print-Friendly** - Clean black-and-white print layout for desk reference
- **Mobile Responsive** - Works on phones, tablets, and desktops
- **Single File** - No dependencies, no build step, no server required

## Quick Start

Open `index.html` in any browser. That's it.
